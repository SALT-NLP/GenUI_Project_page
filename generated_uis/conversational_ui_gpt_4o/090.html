These algorithms learn from labeled data (inputs paired with correct outputs).

1. Linear Regression

  • Goal: Predict continuous values (e.g., house price).
  • How: Fits a straight line (or hyperplane) through data.

2. Logistic Regression

  • Goal: Classify into two (or more) categories.
  • How: Uses the logistic function to output probabilities.
  • Example: Will it rain? (Yes/No)

3. Decision Tree

  • Goal: Classification or regression.
  • How: Builds a tree where each node asks a question (like "Is age > 30?").
  • Output: Leaf nodes give final decision.
  • Advantage: Interpretable rules.

4. Random Forest

  • Goal: Boost accuracy using multiple trees.
  • How: Trains many decision trees on random subsets and averages the results.
  • Advantage: Handles overfitting better.

5. Support Vector Machine (SVM)

  • Goal: Find the best boundary (hyperplane) between classes.
  • How: Maximizes margin between classes, can use kernels for non-linear data.

6. K-Nearest Neighbors (KNN)

  • Goal: Classify based on proximity.
  • How: Look at the k closest training points and choose the majority class.
  • No Training Phase: Stores all data.

  • Goal: Group data into k clusters.

2. Hierarchical Clustering

  • Goal: Build a tree of clusters.
  • How: Merge or split clusters based on distance.

3. Principal Component Analysis (PCA)

  • Goal: Reduce dimensions.
  • How: Finds directions (components) that capture the most variance.
