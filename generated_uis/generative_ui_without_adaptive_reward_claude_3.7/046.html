Important: Credentials will be stored securely as encrypted secrets What credentials are needed?

AWS: Access Key ID and Secret Access Key

Azure: Service Principal or Publish Profile

GCP: Service Account Key

Heroku/Netlify/Vercel: API Token

FTP/SSH: Username and Password/SSH Key

Docker/Kubernetes: Registry credentials

Please enter deployment credentials
Command to execute for deployment Common examples

AWS S3: aws s3 sync ./dist s3://my-bucket/

Docker: docker push username/image:tag
