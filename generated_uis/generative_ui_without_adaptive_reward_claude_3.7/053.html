Interactive Practice Problem

Solving Quadratic Equations

Solve the quadratic equation: x² - 5x + 6 = 0

x = 2 and x = 3
x = -2 and x = -3
x = 2 and x = -3
x = -2 and x = 3

Hint:

Try factoring the quadratic expression. Look for two numbers that multiply to give 6 and add to give -5.

Solution:

x² - 5x + 6 = 0

Factoring: (x - 2)(x - 3) = 0

Therefore, x = 2 or x = 3
